R2 Media Offload is a WordPress plugin that offloads your media library to R2-compatible object storage for cost-effective and efficient media management.

== Description ==
R2 Media Offload helps WordPress users offload their media library to R2-compatible storage solutions using an S3-compatible API. This reduces server load and improves media serving efficiency.

**Features:**
- Automatic media uploads
- Migrate existing media
- Customizable settings for bucket configuration
- Delete local media option

== Installation ==
1. Download the plugin.
2. Upload to the `wp-content/plugins/` directory.
3. Activate via WordPress Admin Plugins page.
4. Navigate to **Settings > R2 Media Offload** to configure.
